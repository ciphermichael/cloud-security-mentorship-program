Take a deliberately vulnerable Docker image. Apply all CIS Docker Benchmark Level 1 hardening steps. Write 5 custom Falco rules (e.g. shell in container, /etc/shadow read, cron modification). Build a CI pipeline that blocks images with CRITICAL CVEs.
