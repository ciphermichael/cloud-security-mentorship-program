## 📝 Weekly Assignment

Build a UEBA engine in Python. Process simulated CloudTrail logs for 50 users. Compute per-user baselines (normal hours, typical API calls, usual regions). Flag: users active outside baseline hours, API call volume 3σ above mean, access from new geolocations.
